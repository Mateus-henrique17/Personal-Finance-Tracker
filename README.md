# Financial Dashboard

Aplicação front-end desenvolvida em JavaScript puro para gerenciamento de transações financeiras. O projeto permite cadastrar transações de entrada e saída, exibir o histórico dinamicamente na interface e calcular automaticamente o saldo atual.

A proposta inicial era criar um dashboard de controle financeiro, mas o projeto evoluiu para uma aplicação focada na experiência de cadastro e visualização de transações em tempo real.

---

## Funcionalidades

- Cadastro de transações
- Exibição dinâmica do histórico
- Identificação visual de entradas e saídas
  - Entrada → destaque em verde
  - Saída → destaque em vermelho
- Cálculo automático do saldo atual
- Persistência local dos dados
- Edição de transações cadastradas

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- ES Modules

---

## Estrutura do projeto

O projeto foi desenvolvido utilizando uma arquitetura modular com ES Modules, mantendo a separação de responsabilidades entre as funcionalidades da aplicação e facilitando a manutenção do código.

A interface é atualizada dinamicamente a partir das interações do usuário, sem utilização de frameworks ou bibliotecas externas.

---

## Conceitos aplicados

- Manipulação de DOM
- Modularização com ES Modules
- Atualização dinâmica de interface
- Persistência de dados no navegador
- Organização de estado da aplicação
- Estruturação de componentes em JavaScript puro
